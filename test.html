Generate Playlist

Are you annoyed that Spotify's Release Radar only shows one song per artist? Are you missing songs or albums by artists you love? Use "Extended Release Radar" to fix this! Click the button below to generate your weekly playlist of every song by every artist you follow.

Important notes: This application will generate every song released in the last week by the artists you follow into a playlist called "Extended Release Radar." If you rerun the program, it will override the playlist. You must follow an artist on Spotify for their songs to appear. Songs may be duplicated if you follow two artists featured on the same song, or if the artist releases more than one version of a song (ie. explicit and clean). Also, very occasionally, songs may be missing if the artist is featured weirdly.

The playlist is not generated automatically. You must click the button every time you want a new playlist. Also, generating the playlist may take a few minutes depending on how many artists you follow.

About Us

This project started out of the Hackathon PackHacks in March 2021. Details about the first version of the project is available here.

The program works by using a python script that uses the Spotify API and the Spotipy API to generate the playlist. Then, Flask is used to run the project. W3School was used to facilate the creation of the HTML website.
